# Bezva Veci

## Pouziti

```sh
npx knihovna-cli <command> [params]
```

Alternativne je mozne `CLI` nainstalovat na stroj pomoci prikazu:

```sh
yarn global add knihovna-cli

# neho

npm install -g knihovna-cli
```

Pokud je nainstalovany `CLI` globalne, pouziva se

```sh
knihovna <command> [params]
```

### Dostupne `command`

#### init

```sh
npx knihovna
```

Dostupne sablony projektu

| Projekt                      | command                | initializace NPX                                            | inicializace globalni instalace                     | Popis |
| ---------------------------- | ---------------------- | ----------------------------------------------------------- | --------------------------------------------------- | ----- |
| Produkcni aplikace v ReactJs | `react-production-app` | `npx knihovna-cli init react-production-app <project-name>` | `knihovna init react-production-app <project-name>` |       |
